# DataVet 🔷
**Vet your data before it enters your pipeline**

A free browser-based data quality tool for ops, finance, and data teams.
No installation. No code. Upload a CSV and get instant quality analysis.

## What it does
- Detects inconsistent values across columns (M / Male / Mal → Male)
- Flags missing data, outliers, duplicates and date format issues
- Fix values with one click — no Excel macros, no Python scripts
- Export a clean CSV with a full corrections log
- Pattern cleaner with regex presets for dates, phones, casing

## How to use
👉 [Launch DataVet](https://YOUR-GITHUB-PAGES-URL)

1. Click **Load Demo** to see it in action
2. Or upload your own CSV file
3. Fix issues using the Cleaning and Value Mapping tabs
4. Export your cleaned file
